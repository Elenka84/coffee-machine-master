

## План: Добавить переключатель темы (день/ночь)

В шапке сайта рядом с кнопкой «Оставить заявку» добавлю иконку-переключатель темы (солнце/луна).

### Что сделаю

1. **Подключить `next-themes`** (уже популярный стандарт для shadcn) — установлю пакет и оберну приложение в `ThemeProvider` в `src/App.tsx`.

2. **Создать компонент `src/components/ThemeToggle.tsx`** — кнопка-иконка (`variant="ghost"`, `size="icon"`) с иконками `Sun`/`Moon` из lucide-react, которая переключает классы `light`/`dark` на `<html>`.

3. **Вставить `<ThemeToggle />` в шапку** `src/pages/Index.tsx` — между телефоном и кнопкой «Оставить заявку», виден на всех разрешениях (включая мобильные).

4. **Тёмная тема уже настроена** в `src/index.css` (блок `.dark { ... }` с кофейной палитрой) — дополнительных правок CSS не требуется. Проверю, что плавающие элементы (мобильная панель звонка, WhatsApp-кнопка) корректно смотрятся на тёмном фоне; при необходимости подправлю токены.

5. **Сохранение выбора** — `next-themes` автоматически запоминает тему в `localStorage` и поддерживает системную тему по умолчанию.

### Файлы

- `package.json` — добавить `next-themes`
- `src/App.tsx` — обернуть в `ThemeProvider` (`attribute="class"`, `defaultTheme="light"`)
- `src/components/ThemeToggle.tsx` — новый файл
- `src/pages/Index.tsx` — вставить кнопку в `<header>`
